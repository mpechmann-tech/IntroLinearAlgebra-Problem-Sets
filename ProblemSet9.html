Problem 1 

Suppose that x1=2,x2=−1,x3=3,x4=0 is a solution of the nonhomogeneous system A⁢𝐱=𝐛, and that the solution set of the homogeneous system A⁢𝐱=0 is given by

x1=3⁢s−2⁢t,x2=−s+4⁢t,x3=s,x4=t,

where s,t∈ℝ.

  1. (a)

    Find a vector form of the general solution of A⁢𝐱=𝟎.

  2. (b)

    Find a vector form of the general solution of A⁢𝐱=𝐛.

Problem 2

Problem 12 *

Problem 13 *

Let A be a square matrix. Determine whether each statement is true or false.

  1. (i)

    If A⁢𝐱=λ⁢𝐱 for some nonzero scalar λ, then 𝐱 is an eigenvector of A. (False)

  2. (ii)

    If λ is an eigenvalue of a matrix A, then the linear system (λ⁢I−A)⁢𝐱=0 has only the trivial solution. (False)

  3. (iii)

    If λ is an eigenvalue of a matrix A, then the eigenspace of A corresponding to λ is the set of eigenvectors of A corresponding to λ. (False)

  4. (iv)

    The eigenvalues of a matrix A are the same as the eigenvalues of the reduced row echelon form of A. (False)

  5. (v)

    If 0 is an eigenvalue of a matrix A, then the columns of A are linearly independent. (False)

  6. (vi)

    If the characteristic polynomial of a matrix A is λ2+1, then A is invertible. (True)
