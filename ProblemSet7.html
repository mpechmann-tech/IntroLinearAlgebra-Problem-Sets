Problem 7 

To answer the questions below, consider first the following Definition and Remark:

Definition 1 (Vector Space).

Let V be a nonempty set together with two operations:

  • •

    addition: +:V×V→V, (𝐮,𝐯)↦𝐮+𝐯,

  • •

    scalar multiplication: ⋅:𝔽×V→V, (λ,𝐯)↦λ⁢𝐯,

where 𝔽 is a field (e.g. 𝔽=ℝ). Then V is called a vector space over 𝔽 if the following axioms hold for all 𝐮,𝐯,𝐰∈V and all λ,μ∈𝔽:

  1. 1.

    If 𝐮,𝐯∈V, then 𝐮+𝐯∈V (closure under addition)

  2. 2.

    𝐮+𝐯=𝐯+𝐮 (commutativity)

  3. 3.

    (𝐮+𝐯)+𝐰=𝐮+(𝐯+𝐰) (associativity)

  4. 4.

    There exists an element 𝟎∈V such that 𝐯+𝟎=𝐯 for all 𝐯∈V (additive identity)

  5. 5.

    For each 𝐯∈V, there exists an element −𝐯∈V such that 𝐯+(−𝐯)=𝟎 (additive inverse)

  6. 6.

    If λ∈𝔽 and 𝐮∈V, then λ⁢𝐮∈V (closure under scalar multiplication)

  7. 7.

    λ⁢(𝐮+𝐯)=λ⁢𝐮+λ⁢𝐯 (distributivity over vector addition)

  8. 8.

    (λ+μ)⁢𝐯=λ⁢𝐯+μ⁢𝐯 (distributivity over field addition)

  9. 9.

    (λ⁢μ)⁢𝐯=λ⁢(μ⁢𝐯) (compatibility of scalar multiplication)

  10. 10.

    1⁢𝐯=𝐯 (identity element of the field)

Remark 2.

The element 𝟎 (see axiom 4) is also called the neutral element of vector addition, since adding it to any vector leaves the vector unchanged. Similarly, the scalar 1∈𝔽 (see axiom 10) is the neutral element of scalar multiplication, since 1⁢𝐯=𝐯 for all 𝐯∈V.

Let V be the set of all ordered pairs of real numbers (i.e. V={(x,y):x,y∈ℝ}) and consider the following addition and scalar multiplication operations on 𝐮=(u1,u2) and 𝐯=(v1,v2):

𝐮+𝐯=(u1+v1,u2+v2),k⁢𝐮=(0,k⁢u2).

Answer the following questions:

  1. (a)

    Compute 𝐮+𝐯 and k⁢𝐮 for 𝐮=(−1,2), 𝐯=(3,4), and k=3.

  2. (b)

    In words, explain why V is closed under addition and scalar multiplication.

  3. (c)

    Since addition on V is the standard addition operation on ℝ2, certain vector space axioms hold for V because they are known to hold for ℝ2. Which axioms are they?

  4. (d)

    Show that axioms 7, 8, and 9 hold.

  5. (e)

    Show that axiom 10 fails, and hence that V is not a vector space under the given operations.

Problem 8 

To answer part (a) - (c) below, consider first the following Theorem:

Theorem 3 (Subspace Test).

Let V be a vector space (over ℝ), and W⊆V such that W≠∅. Then W is a subspace of V if and only if the following two conditions are satisfied:

  1. (i)

    If 𝐮,𝐯∈W, then 𝐮+𝐯∈W.

  2. (ii)

    If k∈ℝ and 𝐮∈W, then k⁢𝐮∈W.

Answer the following questions:

  1. (a)

    Let W be a subspace of a vector space V. Must W contain the zero vector 𝟎 (that is, the neutral element of vector addition)? Explain why or why not.

  2. (b)

    Using the subspace test, determine which of the sets are subspaces of ℝ3 (over ℝ):

    1. (a)

      W={(a,0,0):a∈ℝ}.

    2. (b)

      W={(a,1,1):a∈ℝ}.

    3. (c)

      W={(a,b,c):b=a+c}.

  3. (c)

    Using the subspace test, determine which of the sets are subspaces of Mn⁢n (over ℝ) (Mn⁢n is the set of all n×n-matrices whose entries are real numbers):

    1. (a)

      The set of all diagonal n×n-matrices.

    2. (b)

      The set of all n×n matrices A such that det(A)=0.

    3. (c)

      The set of all n×n matrices A such that tr⁢(A)=0.
