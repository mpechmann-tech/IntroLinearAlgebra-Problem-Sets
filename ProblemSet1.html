Problem 1

Determine whether the following equations are linear (that is, linear in x1,x2, and x3) and whether they are homogeneous.

  1. (a)

    x1+5⁢x2−2⁢x3=1

  2. (b)

    x1+3⁢x2+x1⁢x3=2

  3. (c)

    x1=−7⁢x2+3⁢x3

  4. (d)

    x1−2+x2+8⁢x3−5=0

  5. (e)

    x13/5−2⁢x2+x3=4

  6. (f)

    π⁢x1−2⁢x2+13⁢x3=71/3

Problem 3

In each part, find the augmented matrix corresponding to the given linear system.

  1. (a)

    3⁢x1−2⁢x2=−1

    4⁢x1+5⁢x2=3

    7⁢x1+3⁢x2=2

  2. (b)

    2⁢x1+2⁢x3=1

    3⁢x1−x2+4⁢x3=7

    6⁢x1+x2−x3=0

  3. (c)

    x1+2⁢x2−x4+x5=1

    3⁢x2+x3−x5=2

    x3+7⁢x4=1

  4. (d)

    x1=1

    x2=2

    x3=3

Problem 4

2⁢x1−4⁢x2−x3=1

x1−3⁢x2+x3=1

3⁢x1−5⁢x2−3⁢x3=1

  1. (a)

    (3,1,1)

  2. (b)

    (3,−1,1)

  3. (c)

    (13,5,2)

Problem 5

In each part, determine whether the matrix is in row echelon form (ref), reduced row echelon form (rref), or neither.

  1. (a)

    [100010001]

  2. (b)

    [100010000]

  3. (c)

    [010001000]

  4. (d)

    [10310124]

  5. (e)

    [12030001100000100000]

  6. (f)

    [000000]

  7. (g)

    [1−7550132]

Note: A matrix that is in reduced row echelon form (rref) satisfies all the conditions of row echelon form (ref), so it is always also in ref.
